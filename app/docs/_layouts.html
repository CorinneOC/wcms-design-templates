The layout container

Wraps the layout. The only HTML elements outside of this element are the accessibility links.

#container {
	@include outer-container();
}

The layout row

The row does as it says. Each ‘row’ of content is cleared and and sets the flex property so it’s direct descendants will flex in the right direction.
